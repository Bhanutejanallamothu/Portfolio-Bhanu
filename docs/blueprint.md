# **App Name**: Bhanu Teja Portfolio

## Core Features:

- Profile Showcase: Display Nallamothu Bhanu Teja's personal information, skills, and resume download link.
- Project Gallery: Showcase a selection of Nallamothu Bhanu Teja's projects with descriptions, tech stacks, and links to live demos and repositories.
- Experience Timeline: Highlight Nallamothu Bhanu Teja's work and academic history in a clear and interactive timeline format.
- Contact Form Submission: Allow visitors to send messages to Nallamothu Bhanu Teja directly through a contact form.
- Theme Toggling: Allow users to switch between light and dark themes with persistent local storage.

## Style Guidelines:

- Primary color: Slate blue (#778DA9) to evoke trust and technical expertise.
- Background color: Off-white (#F0F0F0) for the light mode and dark gray (#1B262C) for dark mode to provide comfortable readability.
- Accent color: Soft peach (#E0E5EC) used for highlights and interactive elements to draw attention without being distracting.
- Headline font: 'Space Grotesk', sans-serif, for a modern and tech-oriented feel. Body font: 'Inter', sans-serif, for readability.
- Use simple, outline-style icons to represent different skills and categories, maintaining a clean and modern aesthetic.
- Implement a mobile-first, responsive grid layout to ensure optimal viewing experience across various devices. Use clear visual hierarchy to guide users through the content.
- Employ subtle animations and transitions using Framer Motion for interactive elements, hover states, and page transitions to enhance user experience without overwhelming them.
